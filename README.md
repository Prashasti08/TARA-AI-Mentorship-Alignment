# **TARA: GenAI Mentorship Chatbot**

## **Project Summary**

TARA (Transformative AI Resource for Advancement) is an AI-powered mentorship chatbot built for U.S. graduate students seeking intentional personal and professional growth. It delivers tailored career strategy, skill-building support, and structured guidance in real time. Designed to turn goals into action, TARA helps students make smarter moves, faster.
